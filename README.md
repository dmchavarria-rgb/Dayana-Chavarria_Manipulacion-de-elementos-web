# Maniculacion de elementos web
# Descripción
Este repositorio contiene el proyecto desarrollado en el Laboratorio de DOM de la asignatura Fundamentos de Sistemas Web.
El trabajo consiste en una aplicación web que permite el registro dinámico de estudiantes, integrando funcionalidades de agregar, editar, eliminar, buscar y filtrar registros, además de visualizar la información en formato JSON mediante un modal.

El repositorio incluye:

Archivos del proyecto (index.html, app.js).

Informe académico en formato PDF con capturas y explicación.

Archivo README.md con la documentación básica del trabajo.

# Tecnologías utilizadas
HTML5 – estructura de la aplicación.

CSS3 / Bootstrap 5 – estilos y componentes visuales.

JavaScript – manipulación del DOM, eventos y lógica del sistema.

SweetAlert2 – confirmaciones visuales y mensajes interactivos.

# Forma de ejecución
Clonar o descargar el repositorio.

Abrir el archivo index.html en cualquier navegador web moderno.

Interactuar con el formulario y probar las funcionalidades:

Registrar estudiantes.

Editar y eliminar registros con confirmaciones visuales.

Visualizar información en formato JSON mediante el modal.

Usar búsqueda y filtro por carrera.

Probar el contador dinámico y la limpieza total de registros.
